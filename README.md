## 변수

## 변수 작성법
* `변수선언키워드 생성변수명;` -> 빈 저장소 생성
    * `var box;` : 박스 생성
* `변수선언키워드 생성변수명 대입연산자 대입값;`
    * `var box = 1;` : 박스 생성, 1 대입 // 이미 박스 생성 되었을시는 사용불가(var)
* `생성변수명 대입연산자 대입값;`
    * `box = 1;` : 박스에 1을 대입 // 박스 생성 선행 필수
* 변수선언키워드 : var, let, const
* 생성변수명 - 영문, 숫자조합(숫자는 영문 뒤로)
## 변수 작성시 주의사항
* 변수의 이름은 문자(a~z, A~Z) 숫자(0~9), 달러($), 밑줄(_)이 포함
* 변수의 이름은 숫자로 시작할 수 없음
* 변수의 이름은 대시(-)나 마침표(.) 기호를 사용할 수 없음
* 키워드나 예약어는 사용할 수 없음
    * 키워드, 예약어 종류 : `delete, this, false, true, if, default, var, new, class` 등
* 모든 변수는 대소문자 구별 (score=/Score)
* 변수가 저장할 정보의 종류를 표현하는 의미있는 단어 사용!
## 변수 표기법 종류
* 캐멀 표기법 : newName, webSiteCode (두번째 단어부터 대문자 사용)
* 파스칼 표기법 : NewName, WebSiteCode (각단어 첫글자 대문자 사용)
* 밑줄 표기법 : new_namem, web_site_code (단어 사이 언더바)
## 객체, 속성, 메소드(함수) 작성법
* `객체;` -> 객체 읽음
* `객체.속성;` -> 객체의 속성 읽음
* `객체.메서드();` -> 객체를 메서드 방법으로 실행함
* `객체.속성.메서드();`
* `객체.속성.속성.메서드();`
* `객체.메서드().속성;` -> 객체는 무조건 제일 앞, 메서드는 반드시 뒤에 괄호 포함. 그외 순서 상관없음
* `객체.메서드(메서드());`

## 산술연산자 (피연산자가 숫자인 경우)
* `a + b` : 더하기 // 두개의 피연산자를 더한다.
* `a - b` : 빼기 // 왼쪽 피연산자에서 오른쪽 피연산자를 뺀다. **사용시 문자형->숫자형으로 자동 형변환**
* `a * b` : **곱하기** // 두 개의 피연산자를 곱한다. **문자형->숫자형으로 자동 형변환**
* `a / b` : **나누기** // 왼쪽의 피연산자를 오른쪽 피연산자로 나눈 몫이다. **문자형->숫자형으로 자동 형변환**
* `a % b` : **나머지** // 왼쪽의 피연산자를 오른쪽 피연산자로 나눈 나머지이다.
* `a ** b` : 거듭제곱 // 왼쪽의 피연산자의 오른쪽 피연산자의 값만큼 곱한 값이다.
## 함수 종류
* `prompt('질문')` : 질문답변 **창** 띄우는 함수
    * `input` 태그로 입력받는 사용자입력값도 모두 `문자형`으로 처리
    * 쇼핑몰의 수량 `input`도 문자로 기본적으로 인식됨
* `Number(숫자로변환하고싶은데이터 또는 함수)`
    * `prompt(Number('질문'))` : **X** prompt 질문 입력 위치에 다른 함수 작성
    * `Number(prompt('질문'))` : **O** Number괄호안은 변환대상 prompt함수 작성가능
* `toLocaleString('나라-언어')` : 숫자를 특정 국가의 기호에 맞게 문자열로 변경 표기 (예)`toLocaleString('ko-kr')`
## 복합대입연산자
* 코드의 간결함을 위해 사용한다. 사칙연산 모두 복합대입연산자 적용이 가능하다.
* `a += b` : a = a + b // 기존 a값에 b를 더하여 a에 대입한다.
* `a -= b` : a = a - b // 기존 a값에 b를 빼서 a에 대입한다.
* `a *= b` : a = a * b // 기존 a값에 b를 곱해서 a에 대입한다.
* `a /= b` : a = a / b // 기존 a값에 b를 나눠서 a에 대입한다.
* `a %= b` : a = a % b // 기존 a값에 b를 나눈 나머지값을 a에 대입한다.

## function 함수의 종류
* function : `함수 선언` 시 함수라는 것을 나타내는 키워드
* 함수이름 : 개발자가 만든 함수의 `의미있는 이름`을 작성
* 전달인자 : 매개변수, 인수 등 여러가지 용어를 사용한다, `함수 내에 데이터를 전달`하는 변수명. 전달 데이터가 만을 땐 객체 타입으로 전달하기도 한다.
* 중괄호{} : 함수 영역의 시작과 끝. 함수의 `반복적인 실행 내용`을 작성.

## 함수 생성
1. 반복출력하고 싶은 대상은 함수로 생성하기
    * `function 함수명 () {}`
2. 위 함수 호출 시 마다 바꾸고 싶은 값이 있다면 그 값 개수만큼 매개변수로 만들기
    * `함수명();` //호출
    * `function 함수명 (){반복출력명령어}` // 바뀌는 값이 없을때
    * `function 함수명 (매개변수){반복출력명령어}` //바뀌는 값이 1개일때
    * `function 함수명 (매개1)(매개2){반복출력명령어}` //바뀌는 값이 2개일때
3. 함수 호출 위치에 2에서 생성한 매개변수에 전달하고 싶은 값 입력하기
    * `함수명(매개변수전갈값);`
    * `function 함수명 (매개변수){매개변수가 들어간 반복출력값;}`
