## 변수
* 프로그래밍의 첫번째 시작작업은 필요한 데이터들을 반드시 **변수(저장소)**로 저장해야 한다.
## 변수 작성법
* `변수선언키워드 생성변수명;` -> 빈 저장소 생성
    * `var box;` : 박스 생성
* `변수선언키워드 생성변수명 대입연산자 대입값;`
    * `var box = 1;` : 박스 생성, 1 대입 // 이미 박스 생성 되었을시는 사용불가(var)
* `생성변수명 대입연산자 대입값;`
    * `box = 1;` : 박스에 1을 대입 // 박스 생성 선행 필수
* 변수선언키워드 : var, let, const
* 생성변수명 - 영문, 숫자조합(숫자는 영문 뒤로)
## 변수 작성시 주의사항
* 변수의 이름은 문자(a~z, A~Z) 숫자(0~9), 달러($), 밑줄(_)이 포함
* 변수의 이름은 숫자로 시작할 수 없음
* 변수의 이름은 대시(-)나 마침표(.) 기호를 사용할 수 없음
* 키워드나 예약어는 사용할 수 없음
    * 키워드, 예약어 종류 : `delete, this, false, true, if, default, var, new, class` 등
* 모든 변수는 대소문자 구별 (score=/Score)
* 변수가 저장할 정보의 종류를 표현하는 의미있는 단어 사용!
## 변수 표기법 종류
* 캐멀 표기법 : newName, webSiteCode (두번째 단어부터 대문자 사용)
* 파스칼 표기법 : NewName, WebSiteCode (각단어 첫글자 대문자 사용)
* 밑줄 표기법 : new_namem, web_site_code (단어 사이 언더바)
## 객체, 속성, 메소드(함수) 작성법
* `객체;` -> 객체 읽음
* `객체.속성;` -> 객체의 속성 읽음
* `객체.메서드();` -> 객체를 메서드 방법으로 실행함
* `객체.속성.메서드();`
* `객체.속성.속성.메서드();`
* `객체.메서드().속성;` -> 객체는 무조건 제일 앞, 메서드는 반드시 뒤에 괄호 포함. 그외 순서 상관없음
* `객체.메서드(메서드());`
## 데이터타입 구분 함수 'typeof()'
1. 특정 데이터타입을 담아둔 변수를 생성
    * `let num =  1;`
    * `let str =  '1';`
2. 해당 변수의 데이터타입을 구분하기 위해 typeof 함수 확인
    * console.log(typeof(num)); // Number 숫자
    * console.log(typeof(str)); // String 문자
## 산술연산자 (피연산자가 숫자인 경우)
* `a + b` : 더하기 // 두개의 피연산자를 더한다.
* `a - b` : 빼기 // 왼쪽 피연산자에서 오른쪽 피연산자를 뺀다. **사용시 문자형->숫자형으로 자동 형변환**
* `a * b` : **곱하기** // 두 개의 피연산자를 곱한다. **문자형->숫자형으로 자동 형변환**
* `a / b` : **나누기** // 왼쪽의 피연산자를 오른쪽 피연산자로 나눈 몫이다. **문자형->숫자형으로 자동 형변환**
* `a % b` : **나머지** // 왼쪽의 피연산자를 오른쪽 피연산자로 나눈 나머지이다.
* `a ** b` : 거듭제곱 // 왼쪽의 피연산자의 오른쪽 피연산자의 값만큼 곱한 값이다.
